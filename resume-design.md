# Modern PDF Resume Design - Drishya E

## Design Philosophy

### Visual Language
- **Minimalist Brutalism**: Clean lines with purposeful geometric elements
- **Editorial Sophistication**: Inspired by modern design publications
- **Technical Precision**: Reflecting SharePoint development expertise
- **Human-Centered**: Warm accessibility despite technical focus

### Color Palette
- **Primary**: Warm Charcoal (#2D3748) - Professional depth
- **Secondary**: Soft Sage (#68D391) - Growth and innovation  
- **Accent**: Warm Amber (#F6AD55) - Energy and creativity
- **Background**: Pure White (#FFFFFF)
- **Text**: Deep Charcoal (#1A202C) for high contrast readability

### Typography
- **Display Font**: Canela (Bold) - For name and section headings
- **Body Font**: Suisse Int'l (Regular/Medium) - For content
- **Monospace**: JetBrains Mono - For technical skills

## Layout Structure

### Page Specifications
- **Format**: A4 (210mm × 297mm)
- **Margins**: 20mm all sides
- **Columns**: Single column with strategic white space
- **Grid**: 12-column invisible grid for alignment

### Section Hierarchy
1. **Header**: Name, title, contact info with availability indicator
2. **Professional Summary**: Concise expertise overview
3. **Technical Skills**: Categorized proficiency display
4. **Professional Experience**: Timeline with key achievements
5. **Education**: Academic background
6. **Languages**: Multilingual capabilities

## Print Optimization

### Color Considerations
- **CMYK Safe**: All colors optimized for print
- **Contrast Ratios**: Minimum 4.5:1 for accessibility
- **Black Text**: Pure black for maximum readability
- **Background**: Pure white for clean appearance

### Typography Optimization
- **Font Sizes**: Minimum 10pt for body text
- **Heading Hierarchy**: Clear size differentiation
- **Line Length**: 45-75 characters per line
- **Font Embedding**: Subset fonts for file size

### Layout Stability
- **Fixed Positioning**: No relative positioning
- **Page Breaks**: Strategic section breaks
- **Image Handling**: Vector graphics where possible
- **File Size**: Optimized for email sharing
